API de catálogo de bolos e votações

  • GET: /api/v1/cakes

    Retorna uma listagem com os dados de todos os bolos

  • GET: /api/v1/cakes/{id}

    Parametros da Requisição
    id: identificação do bolo

    Retorna os dados de um bolo específico

  • POST: /api/v1/cakes

    Corpo da Requisição
    Formato: application/json

    {
        "name": "Plum Cake",
        "picture": "4572.jpg",
        "category": "fruit_cake",
        "comments": 0,
        "upvotes": 2
    }

    Inclui os dados de um novo bolo

  • PATCH: /api/v1/cakes/{id}

    Parametros da Requisição
    id: identificação do bolo

    Corpo da Requisição
    Formato: application/json

    {
        "comments": 0,
        "upvotes": 2
    }

    Atualiza algumas informações de um bolo existente

  • PUT: /api/v1/cakes/{id}

    Parametros da Requisição
    id: identificação do bolo

    Corpo da Requisição
    Formato: application/json

    {
        "name": "Plum Fuit Cake",
        "picture": "4572.png",
        "category": "fruit_cake",
        "comments": 1,
        "upvotes": 4
    }

    Atualiza todas as informações de um bolo existente

  • DELETE: /api/v1/cakes/{id}

    Parametros da Requisição
    id: identificação do bolo

    Exclui os dados de um bolo específico
